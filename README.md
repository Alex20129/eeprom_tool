# EEPROM tool 


## Requirements

- C compiler (GCC recommended)
- CMake for building

## Building from Source

```sh
# Clone the repository
git clone https://github.com/Alex20129/eeprom_tool.git

# Generate build files with CMake
cmake -B build

# Build the project
cmake --build build
```

## Usage

```sh
./build/eeprom_tool [options]
```
